# Existing Plugins

ColdFront ships with several plugins (Django Apps) that can be enabled to
provide additional functionality and integration with other applications.

For details, see the documentation for each plugin below:

- [Slurm](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/slurm)
- [FreeIPA](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/freeipa)
- [LDAP protocol User Search](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/ldap_user_search)
- [Project OpenLDAP](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/project_openldap)
- [Mokey/Hydra OpenID Connect](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/mokey_oidc)
- [iQuota](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/iquota)
- [Open OnDemand](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/ondemand)
- [Open XDMoD](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/xdmod)
- [System Monitor](https://github.com/coldfront/coldfront/tree/master/coldfront/plugins/system_monitor) (example of ways to integrate your own plug-ins)  

Community Supported ColdFront Plugins:  

- [OpenStack Plugin](https://github.com/nerc-project/coldfront-plugin-openstack)  
- [Keycloak User Search](https://github.com/nerc-project/coldfront-plugin-keycloak)  
- [Starfish Plugin](https://github.com/fasrc/sftocf)
